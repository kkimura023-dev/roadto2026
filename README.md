# Road to 2026 ⚽

FIFA World Cup 2026 の試合日程把握と、応援チームの決勝トーナメント分岐をシミュレートできる個人向け旅行計画ツール。

## 機能

### 試合SIMタブ
- タブ内でチームを選択（全48チーム対応）
- 1位 / 2位 / 3位通過のシナリオ別に R32〜決勝までのルートを表示
- 3位通過時は進出可能な R32 の全候補を列挙
- グループリーグ3試合と決勝トーナメントを1画面で確認
- 他11グループは FIFA ランキング（2026年4月版）順に勝ち上がる前提で自動計算
- 各試合の時刻を ET / 現地時間（ET比差分付き）/ JST で表示
- スタジアムごとの Google Maps リンク
- 1クリックで Google カレンダー登録
- 旅程サマリ（都市・日付の一覧）

### カレンダータブ
- 全104試合をカレンダー形式で表示（6月・7月を月ナビゲーション）
- 日付をタップするとその日の試合一覧を表示、さらにタップで詳細展開
- 時刻表示の切り替え：現地時間 / JST
- 会場都市フィルタ（16都市）
- チームシミュレーションモード：選択チームの1位・2位・3位通過時の試合のみ表示
- 確定試合（緑）とシミュレーション試合（橙・TBD）を色で識別

### 共通
- PWA 対応（スマホのホーム画面に追加してアプリのように使える）
- iPhone セーフエリア対応（ノッチ・Dynamic Island 対応）

## ローカル開発

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # dist/ に本番ビルド
```
